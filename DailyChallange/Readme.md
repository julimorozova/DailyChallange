##  Getting Started

### Prerequisites

- macOS with Xcode installed
- iOS 13.0+ simulator or device

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/julimorozova/DailyChallange.git
   ```
2. Open the project in Xcode:
   ```bash
   cd DailyChallange
   open DailyChallange.xcodeproj
   ```
3. Select a simulator and press **Run**.
